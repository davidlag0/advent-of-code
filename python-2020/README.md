## Usage

### Create virtual environment with pipenv
`$ pipenv --python 3.10`

### Install dependencies
`$ pipenv install --dev`

### Run tests in current environment
`$ pipenv run pytest`

### Run tests with tox
`$ tox`

### Run vulnerability tests
`$ pipenv run bandit -r day*`
